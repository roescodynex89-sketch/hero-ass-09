# 🚀 IdeaVault

IdeaVault is a full-stack startup idea sharing platform where users can explore innovative ideas, add their own startup concepts, interact with others through comments, and manage their personal idea workspace securely.

# ✨ Features

- 🔐 Authentication with Better Auth
- 🛡️ JWT Protected Private Routes
- 💡 Add Startup Ideas
- 📄 View All Ideas
- 🔍 Search & Filter Ideas
- 📈 Trending Ideas Section
- 🧠 Detailed Idea View
- 💬 Comment & Interaction System
- 👤 My Ideas Dashboard
- ⚡ Update & Delete Own Ideas
- 📱 Responsive UI Design
